## 美国家庭宽带IP代理的几种主要形态

市面上宣称提供美国家庭宽带 IP 的服务，大概分这么几种：

**1. 真实住宅拉线型**
服务商在美国实际地址租房，以住宅用户身份向电信运营商申请家庭宽带，每个 IP 都是运营商直接分配的静态地址。IP 最纯净，成本最高，价格自然也高一截。

**2. 住宅代理池（Rotating Residential Proxy）**
从大量真实用户手机或电脑上汇聚的共享 IP 池，IP 是轮换的，同一个 IP 可能被很多人用过。价格按流量计费，便宜，但 IP 质量参差不齐。

**3. ISP 代理（静态住宅 IP）**
从运营商批量购买 IP 段，归属显示住宅，但实际挂在数据中心机房。IP 稳定，价格居中，但风控识别率比真实家宽高一些。

**4. 纯数据中心 IP 套住宅标签**
最劣质的一类，本质就是普通机房 IP，换个说法卖。上面那个朋友买的就是这种。

这四种里，真实住宅拉线型的 IP 质量排第一，做需要高 IP 纯净度业务的人应该优先看这个方向。
